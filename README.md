# danielbezerrade-python-powerbi-vendas-lucratividade
Este projeto de Business Intelligence (BI) foi desenvolvido para transformar dados brutos de vendas em insights estratégicos acionáveis. O objetivo principal é fornecer uma análise detalhada da performance comercial, da lucratividade de produtos e do desempenho de clientes, capacitando a tomada de decisões orientada a dados.
